What Each Key File Does
src/lib/firebase.js
Initialises the Firebase app with config from Vite env vars (VITE_FIREBASE_*). Exports auth and db instances used everywhere.
src/lib/firestore.js
Drop-in replacement for storage.js. Same loadData() / saveData() API — but reads/writes to Firestore instead of localStorage. App.jsx changes one import line.
src/components/Auth.jsx
Replaced entirely. New version shows a centred "Sign in with Google" button using Firebase's signInWithPopup(GoogleAuthProvider). No email/password fields.
netlify/functions/parse.js
Replaces server/proxy.js. Netlify serverless function — receives POST with X-Api-Key header, calls Anthropic, returns result. No API key stored on server.
src/tabs/SetupTab.jsx
Gains one new section: "AI Settings" — a password input for the Anthropic API key with a Save button. Saves to Firestore profile.anthropicApiKey.
src/engine/AIParser.js
Reads the Anthropic key from app state (loaded from Firestore) and passes it as X-Api-Key header. URL changes from /api/parse → /.netlify/functions/parse.
